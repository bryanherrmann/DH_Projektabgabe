# DH_Projektabgabe

Um die Datensätze herunterzuladen muss man zuerst https://www.kaggle.com/datasets/gufukuro/movie-scripts-corpus/data besuchen. Ich habe den Datensatz als zip-Datei heruntergeladen. Nachdem die zip entpackt worden ist müssen dann nur noch die Pfade in SkriptSzenenGrafik.R so angepasst werden, dass sie auf die richtigen Ordner verweisen und dann kann das Skript in RStudio ausgeführt werden.
